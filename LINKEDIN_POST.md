LinkedIn Post — SkillCraft Landing Page

I just built a responsive landing page for SkillCraft as part of my web development project (SCT_WD_1).

What I built:
- A modern, responsive landing page with Home, About, Services, and Contact sections.
- A fixed navigation bar that stays visible on scroll, changes style when scrolled, and has hover feedback.
- Smooth-scrolling anchors, mobile-friendly hamburger menu, and a simple contact form handler.

What I learned:
- Practical responsive layout patterns using CSS Grid and Flexbox.
- How to build a small, accessible navigation: sticky header, mobile toggle, and keyboard-friendly links.
- Using minimal JavaScript for UX: smooth scroll, scroll detection, and unobtrusive form handling.

Try it locally: run `python -m http.server 8000` and open http://localhost:8000
